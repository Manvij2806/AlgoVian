# AlgoVian 🚀

AlgoVian is a Design and Analysis of Algorithms (DAA) learning platform built using HTML, CSS, and JavaScript.

## 🌟 Features
- Learn DAA concepts
- Algorithm Visualizer
- Interactive Games
- Leaderboard
- User Signup/Login

## 🛠️ Tech Stack
- HTML
- CSS
- JavaScript
- Node.js (Backend)

## 🚀 How to Run

### Frontend
Open index.html in browser

### Backend
cd backend
npm install
npm start
